# ТУГЕЛ — Жоба іске қосу нұсқаулығы
## (Нөлден бастап, Windows үшін)

---

## 1-ҚАДАМ: Node.js жүктеп алу

1. Браузерде мына сілтемені аш: **https://nodejs.org**
2. Сол жақтағы **"LTS"** батырмасын басып жүктеп ал
3. Жүктелген `.msi` файлды іске қос
4. Барлық жерде **"Next"** → **"Install"** → **"Finish"**

> Тексеру: `Win + R` → `cmd` → `node --version` жаз → `v20.x.x` шықса OK

---

## 2-ҚАДАМ: PostgreSQL жүктеп алу

1. Браузерде аш: **https://www.postgresql.org/download/windows/**
2. **"Download the installer"** сілтемесін бас
3. Ең жаңа нұсқаны (16.x) Windows x86-64 үшін жүктеп ал
4. Жүктелген файлды іске қос
5. Орнату барысында:
   - **Installation Directory** — өзгертпе, Next
   - **Components** — бәрі таңдаулы, Next
   - **Data Directory** — өзгертпе, Next
   - **Password** — `password` деп жаз *(міндетті, осылай жазу керек!)*
   - **Port** — `5432` (өзгертпе), Next
   - **Locale** — өзгертпе, Next
   - **Next** → **Finish**

> Соңында "Stack Builder" сұраса — жабып тастауға болады

---

## 3-ҚАДАМ: Жобаны баптау (БІР РЕТ ҒАНА)

Жоба қалтасын ашыңыз. Ол жерде `setup.bat` файлы бар.

**`setup.bat` файлын тышқанның оң жағымен басып → "Запуск от имени администратора" (Run as administrator)**

Скрипт автоматты түрде:
- Дерекқорды жасайды
- Кестелерді қосады
- Тест деректерін енгізеді
- npm пакеттерін орнатады

---

## 4-ҚАДАМ: Жобаны іске қосу (ЖАЙ АШУ)

**`start.bat`** файлын **қос рет бас** (двойной клик)

Екі терминал ашылады:
- Бірі — Backend (API сервер, порт 4000)
- Бірі — Frontend (сайт, порт 5173)

Содан кейін браузерде аш: **http://localhost:5173**

## Жиі қателер

**"node не является внутренней командой"**
→ Node.js дұрыс орнатылмаған. Компьютерді қайта іске қос, қайталап байқа.

**"psql не является внутренней командой"**
→ PostgreSQL PATH-ке қосылмаған. `setup.bat` өзі табады, бірақ егер қате шықса — PostgreSQL-ді қайта орнат.

**"password authentication failed"**
→ PostgreSQL орнатқанда `password` деп жазбадың. PostgreSQL-ді қайта орнат, пароль: `password`

**Порт 4000 немесе 5173 бос емес**
→ Бұрын ашылған терминалдарды жап да `start.bat` қайта іске қос.
