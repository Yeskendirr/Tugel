# Колледж жабдықтарын түгендеуге арналған веб-қосымша

Колледж оборудованиесін есепке алуға (инвентаризация) арналған веб-қосымша.
Интерфейс толығымен қазақ тілінде.

## Технологиялар

- **Frontend:** React + TypeScript + Vite
- **Backend:** Node.js + Express + TypeScript
- **Дерекқор (БД):** MySQL

## Жоба құрылымы

```
Tugel/
├── backend/      # Node.js API сервер
├── frontend/     # React клиент
└── database/     # SQL схема және бастапқы деректер
```

## Іске қосу

### Backend
```bash
cd backend
npm install
cp .env.example .env   # баптауларды толтыру
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Дерекқор
`database/schema.sql` файлын MySQL-ге импорттаңыз.

## Негізгі мүмкіндіктер (жоспар)

- Пайдаланушыларды авторизациялау (әкімші / қызметкер)
- Жабдықтарды тіркеу, өңдеу, жою
- Кабинеттер мен санаттар бойынша есеп
- Түгендеу актілерін жүргізу
- Есептерді экспорттау
